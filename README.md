# hexv
![Rust](https://img.shields.io/badge/rust-%23000000.svg?style=for-the-badge&logo=rust&logoColor=white)  
hexv is a hex viewer for the terminal.  
The left tab shows hexa and the right tab shows ASCII code.

### Used Library
- clap
- colored

# Example
run: ./hexv [FILE NAME]
